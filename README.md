# pathPRS_by_annotation
pathway-specific polygenic risk scores across SNP-to-gene annotation strategies  


Code used to perform analyses for the manuscript:  

Bazemore K, Iqbal T, Kuzma AB, Grant SFA, Schellenberg GD, Wang LS, Chesi A, Jin J, Naj AC. Functionally informed annotation influences pathway-specific polygenic risk and disease inference in Alzheimer's disease. medRxiv [Preprint]. 2026 May 26:2026.05.25.26353905. doi: 10.64898/2026.05.25.26353905. PMID: 42245020; PMCID: PMC13232378.  

Code for the S-3 annotation strategy can be found at: https://github.com/thewonlab/H-MAGMA
